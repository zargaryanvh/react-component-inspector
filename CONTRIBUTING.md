# Contributing to React Component Inspector

Thank you for your interest in contributing! This project was fully designed by Cursor AI, and we welcome contributions from the community.

## Development Setup

1. Fork the repository
2. Clone your fork
3. Install dependencies: `npm install`
4. Make your changes
5. Test your changes
6. Submit a pull request

## Code Style

- Use TypeScript
- Follow React best practices
- Keep components focused and modular
- Add comments for complex logic
- Ensure all code is development-only (check `NODE_ENV`)

## Testing

While there are no automated tests yet, please:
- Test in a real React application
- Verify it works in development mode
- Confirm it's disabled in production
- Test with different UI libraries (MUI, etc.)

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Update documentation if needed
4. Submit PR with clear description
5. Wait for review

## Questions?

Open an issue or start a discussion!
